DAME-BS — Differentially Private Mean Estimation using Binary Search

Internship - Criteo AI Labs
Repo Docs
Worked on algorithm DAME-BS for mean estimation under user-level Local Differential Privacy (LDP) and implemented it in python. The algorithm has two phases: Localization via private binary search + Estimation via noisy projection & averaging. It also includes theoretical bounds and experiments across distributions.
Summary
  • Localization phase: private binary search to find an interval for the mean.
  • Estimation phase: projection + Laplacian noise and averaging for final estimate.
  • Includes upper bound proofs and empirical evaluation across distributions.
